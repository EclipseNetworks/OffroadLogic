SECURITY POLICY
===============

Protecting the project, one patch at a time.

------------------

Live support status:

Version    Status
-------------------------

Note:
Only supported versions receive security fixes and updates.


REPORTING A VULNERABILITY
------------------------

Found a security issue?
Please report it responsibly.

Process:
1. Submit a private security report to the maintainers
2. Do NOT disclose the issue publicly
3. Wait for review and response
